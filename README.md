# 🚚 Working Hours Logger (v1.0)

Αυτό το project είναι ένα απλό CLI (Command Line Interface) εργαλείο γραμμένο σε Python, σχεδιασμένο για οδηγούς, διανομείς ή επαγγελματίες που χρειάζονται γρήγορη και αξιόπιστη καταγραφή των καθημερινών στοιχείων εργασίας (ώρες, χιλιόμετρα, παραλαβές/παραδόσεις).

Η εφαρμογή χρησιμοποιεί ένα ευανάγνωστο, χρωματικά κωδικοποιημένο περιβάλλον στο τερματικό για ευκολία στη χρήση.

---

## ✨ Βασικά Χαρακτηριστικά

*   **Μηνιαία Αρχεία**: Τα δεδομένα αποθηκεύονται σε ξεχωριστά αρχεία **JSON ανά μήνα** (`ergasia_log_MM_YYYY.json`) για βέλτιστη οργάνωση.
*   **Έξυπνη Εισαγωγή**: Αποτρέπει την καταχώρηση για μελλοντικές ημερομηνίες και εμποδίζει τις διπλές εγγραφές για την ίδια ημέρα.
*   **Αναλυτικά Στατιστικά**: Υπολογίζει και εμφανίζει σύνολα και μέσους όρους για τον επιλεγμένο μήνα.
*   **Καταγραφή Σφαλμάτων**: Τυχόν σφάλματα κατά την εκτέλεση καταγράφονται στο αρχείο `error_log.txt` για μελλοντική ανάλυση.
*   **Μηδενικές Εξαρτήσεις**: Δεν απαιτείται η εγκατάσταση εξωτερικών βιβλιοθηκών.

---

## 🚀 Εκκίνηση

### Προαπαιτούμενα

*   **Python 3.6+**
*   Ένα σύγχρονο Terminal (π.χ. Windows Terminal, PowerShell, Bash) με υποστήριξη **Unicode (UTF-8)** για σωστή εμφάνιση των χρωμάτων και των συμβόλων.

### Εκτέλεση

1.  Αποθηκεύστε το αρχείο `Working_Hours.py` σε έναν φάκελο.
2.  Ανοίξτε το τερματικό σας σε αυτόν τον φάκελο.
3.  Εκτελέστε την παρακάτω εντολή:
    ```bash
    python Working_Hours.py
    ```

## 📋 Λειτουργίες

| Επιλογή | Περιγραφή | Σημειώσεις |
| :--- | :--- | :--- |
| **1. Εισαγωγή νέων δεδομένων** | Καταχωρεί νέα δεδομένα εργασίας για μια συγκεκριμένη ημέρα. | Γίνεται έλεγχος εγκυρότητας της ημερομηνίας και αποτρέπονται οι διπλοεγγραφές. |
| **2. Επεξεργασία δεδομένων** | *(ΥΠΟ ΚΑΤΑΣΚΕΥΗ)* | Μελλοντική λειτουργία για την επεξεργασία ή διαγραφή μεμονωμένων καταγραφών. |
| **3. Προβολή δεδομένων & Στατιστικά** | Εμφανίζει αναλυτικά στατιστικά για τον επιλεγμένο μήνα. | Τα δεδομένα εμφανίζονται ταξινομημένα και περιλαμβάνουν σύνολα και μέσους όρους. |
| **5. Διαγραφή Αρχείου Log (RESET)** | Εμφανίζει οδηγίες για τη χειροκίνητη διαγραφή ενός αρχείου log. | **Προσοχή:** Η διαγραφή είναι οριστική. |
| **0. Κλείσιμο προγράμματος** | Τερματίζει την εφαρμογή. | |

### Στατιστικά Που Υπολογίζονται

*   **Συνολικές Ώρες, Χιλιόμετρα, Παραλαβές & Παραδόσεις** για τον μήνα.
*   **Συνολικός Αριθμός Στοπ** (`Παραλαβές + Παραδόσεις`).
*   **Μέσοι Όροι** (Ώρες, Χλμ, Παραδόσεις, Στοπ) ανά καταγεγραμμένη ημέρα.

## 📂 Δομή Αρχείων & Δεδομένων

*   `Working_Hours.py`: Το κύριο αρχείο του προγράμματος.
*   `ergasia_log_MM_YYYY.json`: Τα αρχεία δεδομένων, ένα για κάθε μήνα/έτος.
*   `error_log.txt`: Αρχείο καταγραφής προειδοποιήσεων και σφαλμάτων.

**Παράδειγμα δομής του JSON:**

```json
[
    {
        "ΗΜΕΡΟΜΗΝΙΑ": "30/10/2025",
        "ΩΡΕΣ": 4.0,
        "ΧΙΛΙΟΜΕΤΡΑ": 49.0,
        "ΠΑΡΑΛΑΒΕΣ": 0,
        "ΠΑΡΑΔΟΣΕΙΣ": 9,
        "ΚΑΤΑΓΡΑΦΗ_ΣΕ": "2025-11-01 16:01:02"
    }
]
```
