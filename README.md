# 🚚 Working Hours Logger (v1.0 Beta)

Αυτό το project είναι ένα απλό CLI (Command Line Interface) εργαλείο γραμμένο σε Python, σχεδιασμένο για οδηγούς, διανομείς ή επαγγελματίες που χρειάζονται γρήγορη και αξιόπιστη καταγραφή των καθημερινών στοιχείων εργασίας (ώρες, χιλιόμετρα, παραλαβές/παραδόσεις).

Τα δεδομένα αποθηκεύονται σε ξεχωριστά αρχεία **JSON ανά μήνα** για βέλτιστη οργάνωση και ταχύτητα.

---

## 🚀 Εκκίνηση του Project

### Προαπαιτούμενα

* **Python 3.x**
* Ένα σύγχρονο Terminal (PowerShell, CMD, Bash) με υποστήριξη **Unicode (UTF-8)** για σωστή εμφάνιση των χρωμάτων και των Emojis.

### Εγκατάσταση και Εκτέλεση

1.  **Clone** (Κλωνοποίηση) του Repository:
    ```bash
    git clone https://github.com/christosk8926/WorkingHourLog.git
    cd WorkingHoursProject
    ```

2.  **Εκτέλεση του Προγράμματος:**
    ```bash
    python Working_Hours.py
    ```

---

## 📋 Λειτουργίες (Features)

| Επιλογή | Περιγραφή | Σημειώσεις |
| :--- | :--- | :--- |
| **1. Εισαγωγή νέων δεδομένων** | Καταχωρεί νέα δεδομένα εργασίας. | Απαιτεί επικύρωση ημερομηνίας, αποτρέπει διπλές καταγραφές. |
| **2. Επεξεργασία δεδομένων** | *(ΥΠΟ ΚΑΤΑΣΚΕΥΗ)* | Προορίζεται για την επεξεργασία ή διαγραφή μεμονωμένων καταγραφών. |
| **3. Προβολή δεδομένων & Στατιστικά** | Εμφανίζει αναλυτικά στατιστικά για τον επιλεγμένο μήνα. | Ζητάει Μήνα (1-12) και Έτος. Τα δεδομένα εμφανίζονται ταξινομημένα. |
| **5. Διαγραφή Αρχείου Log (RESET)** | **Προσοχή!** Διαγράφει το αρχείο JSON του τρέχοντος μήνα. | Χρησιμοποιείται για δοκιμές (Testing) ή ολική διαγραφή δεδομένων. |
| **0. Κλείσιμο προγράμματος** | Κλείνει την εφαρμογή. | |

### Βασικά Στατιστικά Που Υπολογίζονται

Το σύστημα υπολογίζει και εμφανίζει:
* **Συνολικές Ώρες, Χιλιόμετρα, Παραλαβές & Παραδόσεις** για τον μήνα.
* **Συνολικός Αριθμός Στοπ** (`Παραλαβές + Παραδόσεις`).
* **Μέσους Όρους** (Ώρες, Χλμ, Παραδόσεις) ανά καταγραφή.
* **Μέσος Όρος Στοπ ανά Ημέρα** ($Stops/Day$).
